## 方案 1:樂饗購(Level 2 ★★)

**適用期間:** 2026/01/01 ~ 2026/06/30

### 基礎回饋
- 一般消費:0.3%
- 指定購物 / 餐飲消費:3%

### 精選商家回饋(類別)

| 類別 | 回饋 |
|---|---|
| 國內指定百貨(見下方清單) | 3% |
| 國內藥妝 | 3% |
| 國內餐飲 | 3% |
| 麥當勞 | 3% |
| Uber Eats / foodpanda | 3% |

### 補充說明

- **國內指定百貨 3%**:認定依刷卡簽單特店名稱。UNIQLO / GU / Gap / H&M / ZARA / lululemon 等品牌需**於指定百貨內消費**才適用;在品牌直營店刷不算。
- **國內餐飲 3%**:依 MCC code(5811 / 5812 / 5814 / 5462)認定。以下情況**不計入回饋**:
  - 購買餐券(含單筆 NT$10 萬以上之饗賓集團禮券)
  - 設立於百貨 / 商場 / 車站商場 / 機場美食街 / 國道服務區美食街等場所內的餐廳
  - 菸酒販賣商
- **麥當勞 3%**:僅限「樂饗購」方案適用;不適用百貨 / 商場 / 車站 / 機場 / 國道服務區等實體門市。

## 方案 2:玩數位(Level 2 ★★)

### 基礎回饋
- 一般消費:0.3%
- 指定線上消費:3%

### 精選商家回饋(全部 3%)

| 類別 | 商家 |
|---|---|

| 網購平台 | 蝦皮購物、momo 購物網、PChome 24h 購物、小樹購 |

| AI 工具 | ChatGPT、Canva、Claude、Cursor、Duolingo、Gamma、Gemini、Notion、Perplexity、Speak |

### 補充說明

- **Apple 媒體服務**:含 App Store / Apple Music / iCloud 等服務,**不含** Apple Store 硬體交易。
- **PChome 24h 購物**:不含儲值及電子票券。
- **小樹購**:不含電子票券。

## 方案 3:趣旅行(Level 2 ★★)

**適用期間:** 2026/01/01 ~ 2026/06/30

### 基礎回饋
- 一般消費:0.3%
- 指定旅遊 / 交通消費:3%

### 精選商家回饋(全部 3%)

| 類別 | 商家 |
|---|---|
| 海外實體消費 | 含國外餐飲(MCC 5811/5812/5814) |
| 日本指定樂園 | 東京迪士尼樂園、東京華納兄弟哈利波特影城、大阪環球影城(USJ) |
| 國內外交通 | Apple 錢包指定交通卡(SUICA/PASMO/ICOCA)、Uber、Grab、台灣高鐵、台灣大車隊、yoxi、iRent、和運租車、格上租車 |
| 指定航空公司(20 家) | 中華航空、長榮航空、星宇航空、樂桃航空、捷星航空、ANA 全日空、聯合航空、越捷航空、達美航空、卡達航空、台灣虎航、國泰航空、阿聯酋航空、酷航、日本航空、亞洲航空(AirAsia)、新加坡航空、大韓航空、土耳其航空、法國航空 |
| 指定飯店集團 | 星野集團、全球迪士尼飯店、東橫 INN |
| 國內飯店住宿 | 台澎金馬地區民宿、青年旅館、連鎖飯店(依 MCC 認定) |
| 指定旅遊訂房平台(7 家) | KKday、Klook、Agoda、Airbnb、Booking.com、Trip.com、ezTravel 易遊網 |
| 指定旅行社(12 家) | 雄獅旅遊、可樂旅遊、東南旅遊、五福旅遊、燦星旅遊、山富旅遊、長汎假期、鳳凰旅行社、Ezfly 易飛網、永利旅行社、理想旅遊、三賀旅行社 |

### 補充說明

- **海外實體消費**:須為面對面交易(含 Apple Pay / Samsung Pay / Google Pay / Garmin Pay);排除醫療、博奕、機艙/郵輪消費、境外投資平台(如 eToro)、保費等。
- **日本樂園**:含園內商店實體消費及官網/App 購票,**不含** 市區街邊店及附設飯店。
- **航空公司**限航空公司臨櫃、官網或官方 App 購票,**不含** 旅行社、線上旅遊平台;不含機上免稅、機場商店、貴賓室、哩程、套裝行程。
  - 中華航空不含華信;長榮航空不含立榮;大韓航空不含韓亞。
  - 亞洲航空(AirAsia Move)限機票消費,不含旅館/SNAP/叫車/保險等。
- **國內交通**僅限短程叫車,不含長租車、機場接送、代駕、企業長租、訂閱租賃、中古車租賃。
- **Uber** 含國內外搭乘;**Grab** 限搭乘,不含訂閱費。
- **台灣高鐵**限高鐵網站/APP/臨櫃/自動售票機,不含補票、車上推車消費。
- **飯店**:先訂後付/到店付款需於**實際扣款當日**切換為趣旅行方案,否則不適用。
- **Booking.com** 不含租車、景點、活動、計程車。
- **旅行社**限直營,不含加盟店、旗下子公司。
