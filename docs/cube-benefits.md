## 方案 1:樂饗購(Level 2 ★★)

**適用期間:** 2026/01/01 ~ 2026/06/30

### 補充說明

- **國內指定百貨 3%**:認定依刷卡簽單特店名稱。UNIQLO / GU / Gap / H&M / ZARA / lululemon 等品牌需**於指定百貨內消費**才適用;在品牌直營店刷不算。
- **國內餐飲 3%**:依 MCC code(5811 / 5812 / 5814 / 5462)認定。以下情況**不計入回饋**:
  - 購買餐券(含單筆 NT$10 萬以上之饗賓集團禮券)
  - 設立於百貨 / 商場 / 車站商場 / 機場美食街 / 國道服務區美食街等場所內的餐廳
  - 菸酒販賣商
- **麥當勞 3%**:僅限「樂饗購」方案適用;不適用百貨 / 商場 / 車站 / 機場 / 國道服務區等實體門市。
